Czym jest jQuery?

Biblioteka ułatwiająca operacje na drzewie DOM.

Używamy selektorów CSS do wyboru elementów drzewa DOM.

Uniwersalna dla różnych przeglądarek.

Wybieranie elementu drzewa DOM

              
                $('li.list')

Wybieranie elementu drzewa DOM i dodawanie mu klasy

            
              $('li.list').addClass('complete7')

Wybieranie elementu drzewa DOM za pomocą id

              
                $("#first_button")

Wybieranie elementu drzewa DOM za pomocą id i dodawanie mu stylów CSS

              
                $("#first_button").css('background-color', 'yellow')

Zmiana zawartości

              
                $('div:first').text('Tu zmieniam tekst')

Najprostsze wydarzenia

              
                $('li').on('click', function(){
                  $(this).css('color', 'red')
                })
